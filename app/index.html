Skinny Ties

As a developer we are often tasked with creating a website from an image provided too us from a designer. During the first week of school at The Iron Yard we were assigned this task using HTML, CSS, Sip color picker, and the Perfect Pixel tool. Getting things to line up just right was surprisingly challenging and time consuming and it gave me a new appreciation for websites.

Simple Calculator

This simple calculator can add, subtract, multiply, and divide whole numbers. Figuring out the logic in making a calculator work was exhilarating and was my first step into real programming.

Etsy Live

In this project we continued to work with creating a website based on an image, however in this case we also used jQuery for DOM manipulation and we used Ajax with JSON objects to pull in live data from the Etsy API to populate our site. When the live data finally came in and I was able to dig deep into the JSON data and pull out what I needed it was a great feeling of accomplishment.

Other than all of the technical skills such as learning to collaborate on git and dealing with Flickr's Photo API, the project required the element of working as a team and learning how to divide up modular pieces of work so that everyone could contribute.

Around The World

After building a couple of simple contact lists with Backbone and React we were assigned to create an Instagram like clone. We used Backbone to make Models, Collections and Routes for the application. We used React to write all of the components. In this application users can add and edit images stored in the PARSE database. This was one of the more challenging times for me in the course as we went from using only a few files to make a website into using many files to build a modular application.

Bill Splitter

This was a project that I worked on in a group which we used Angular to connect to the server and database created by the Back End students on our team. This was my first ever Hackathon which started after a full day at school on a Friday evening all the way to Sunday afternoon where we gave our sales pitch for our project. While overcoming a short time limit and very little sleep we managed to put together a MVP application to help people who have roommates split the rent and utility bills with as little effort as possible.

Github | Live Site

Five Words

This was our final group project. During this time with the help of another Front End Engineer and a Rails Developer we put together an application for meeting friends based on the matching of simple words. We decided to use Angular for this project and although I had used it in the past it was amazing to learn Angular not through lectures but by simply discovering something that we wanted our app to do and finding a way to make it possible either through Angular or sometimes simply just writing some basic Vanilla JavaScript. While presenting this project at Demo Day and seeing people truly interested in what you made was something I will never forget and with this project I also built a new level of confidence in my ability to teach my self any of the necessary skills to pursue this career
